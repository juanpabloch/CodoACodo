¿Que es la Resonancia Magnética?

La resonancia magnética nuclear, es un método para producir imágenes detalladas de los órganos y tejidos del cuerpo sin la necesidad de usar rayos X o radiación "ionizante". La RM utiliza un poderoso campo magnético, ondas de radiofrecuencia y una computadora para crear imágenes, las cuales sirven para evaluar lesiones, enfermedades o condiciones anormales.

Como prepararse:

Puede utilizar esta información para prepararse para la mayoría de las resonancias magnéticas. La preparación para una resonancia magnética cardíaca o de próstata otros órganos pueden seguir instrucciones especiales.

Implantes

informe a nuestro personal de resonancia magnética si tiene algún implante metálico o electrónico en su cuerpo y traiga la documentación sobre el mismo a su cita.

Contraste

El contraste se administra por vía intravenosa. Se necesitan 3 horas de ayuno mínimo para realizar el estudio con contraste.

Ropa

use ropa cómoda y sin detalles metálicos. En el centro contamos con camisolines si necesita cambiarse.
